Step to run the project 

Install all required Python packages using: pip install -r requirements.txt

Make sure your project folder looks like this:
project-folder/
│
├── app.py
├── selenium_scraper.py
├── bot.py
├── requirements.txt
│
├── templates/
│   └── dashboard.html
│
└── static/
    └── style.css


Configure Review URL
Open app.py and update the review source URL if required:
REVIEW_URL = "https://www.trustpilot.com/review/www.flipkart.com"

Run the Flask Application
Start the web server:python app.py
If successful, you will see:Running on http://127.0.0.1:5000/

Generate Auto Replies
Click “Scrape & Generate Replies”
Reviews will be processed
Sentiment and auto-generated replies will appear in the table
