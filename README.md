Automatic Review System

An AI-based system that analyzes user reviews and automatically generates professional response recommendations through a web-based dashboard.

OBJECTIVES OF THE REVIEW SYSTEM

⦁	To automatically analyze user reviews

⦁	To classify reviews based on sentiment

⦁	To generate appropriate and polite replies

⦁	To reduce manual effort in review management

⦁	To improve response time and consistency

FEATURES OF THE REVIEW SYSTEM

⦁	Sentiment analysis using NLP

⦁	Rule-based and AI-assisted reply generation

⦁	Web-based admin dashboard

⦁	Real-time processing

⦁	No database dependency

⦁	Platform-safe and policy-compliant design


WORKING OF THE REVIEW SYSTEM (STEP-WISE)

⦁	Reviews are collected from public sources or manual input.

⦁	The system processes the review text.

⦁	Sentiment analysis is performed.

⦁	An automated reply is generated.

⦁	The reply is displayed on the dashboard.

⦁	The administrator can review and post the reply.

TECHNOLOGIES USED

1.	Python

2.	Flask

3.	Selenium (for testing on real websites)

4.	Natural Language Processing (TextBlob)

5.	HTML, CSS


LIMITATIONS

⦁	Direct auto-reply on platforms like Google or Shiksha is restricted due to policy rules.

⦁	Dynamic websites may block automated scraping.

⦁	Human approval is required before posting replies on third-party platforms.
