# Automatic-Review-System-
An AI-based system that analyzes user reviews and automatically generates professional response recommendations through a web-based dashboard.

TECHNOLOGIES USED
    
Python

Flask
    
Selenium (for testing on real websites)

Natural Language Processing (TextBlob)

HTML, CSS

WORKING OF THE REVIEW SYSTEM (STEP-WISE)

1.Reviews are collected from public sources or manual input.

2.The system processes the review text.

3.Sentiment analysis is performed.

4.An automated reply is generated.

5.The reply is displayed on the dashboard.

6.The administrator can review and post the reply.
